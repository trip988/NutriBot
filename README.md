# NutriBot - AI Nutrition Assistant

An AI-powered chatbot that tells you the nutritional content 
of any food or drink including calories, protein, carbs, fats, 
vitamins and minerals.

## Live Demo
https://nutribot-byxpcpczdybfq79gpzqwxp.streamlit.app/
The app is hosted on Streamlit Cloud free tier which has a cold start delay of about 30-60 seconds after inactivity. In production this would be solved with paid always-on hosting

## Built With
- Python
- Streamlit
- Groq LLaMA API

## Features
- Calories count
- Protein content
- Carbohydrates and fats
- Vitamins and minerals
- Health tips
